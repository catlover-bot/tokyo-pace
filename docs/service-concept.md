# サービスコンセプト

動的経路は「安全な経路」ではなく、OpenStreetMap属性に基づく比較候補です。標準、階段回避要求、車いすプロファイルをTOKYO PACEの決定的スコアで再評価します。車いすで通れる、必ず通れる、施設を利用・休憩できるとは断定せず、現地の工事、段差、路面、入場条件、営業時間の確認を促します。

休憩候補は`confirmed`（現地確認・座席明示）、`supported`（複数の公式属性）、`possible`（施設の存在のみ）、`estimated`（デモ推定）を分離します。公共施設やバリアフリー掲載施設を自動的に休憩可能とは判定しません。

距離は施設から折れ線までの推定直線距離、またはデモ総距離へ正規化したルート沿い推定距離です。実際の徒歩経路、営業時間、入館条件、着席、給水地点の座席、安全性は保証しません。追加休憩地点は都市設備配置検討の理論値です。

TOKYO PACE は、長時間続けて歩くことが難しい人が、休憩候補・トイレ候補・坂道の情報を比較し、「最後まで歩き切れる可能性の高い経路」を自分で選ぶための支援サービスです。最短経路の代替ではなく、判断材料を増やすことを目的とします。

## 中心指標

経路を休憩機会ごとの連続歩行区間に分け、利用者が設定した「休まずに歩ける時間」と比較します。最大連続歩行時間、次の休憩候補までの最長距離（最長休憩空白）、上限超過時間を区間データから導出し、移動を継続できる条件かを平易な日本語で示します。この判定は経路全体の安全保証ではなく、利用者が比較するための一つの判断材料です。

## MVP の前提

- 対象は新宿駅西口から東京都庁までに限定します。
- 経路・地点・設備属性はすべてプロトタイプ用の推定データであり、現地調査済みではありません。
- 区間距離・区間歩行時間と、それらから導出する移動継続可能性もデモ値です。公式データや実測値ではありません。
- 公衆トイレと車椅子使用者対応情報には新宿区・東京都福祉局の公式オープンデータを使用します。公式掲載は現在利用可能であることや、そこまで通行可能であることを保証しません。
- 安全性、バリアフリー性、設備の利用可否を保証しません。
- 高齢者を主対象に、大きな操作領域、文字情報による凡例、地図外の一覧を提供します。

## 公式情報と推定情報

公式トイレは利用保証や経由地点ではなく、「ルート周辺に公式掲載情報がある候補地点」と表現します。同一建物・同一座標に複数の設備レコードがあっても候補地点数を過大評価せず、詳細から原レコードと出典を確認できるようにします。

公衆トイレ候補と、公共施設内・鉄道駅内の車椅子使用者対応設備情報は評価上も分離します。既定条件は250m以内の公衆トイレ候補だけで成立します。ルート上で公衆トイレ候補の間隔が最も長い区間を「最長トイレ空白」として示しますが、これはデモ折れ線上の推定であり、実際の徒歩距離や利用可否を保証しません。

「ルートから候補まで」は折れ線への推定直線距離です。「ルート上の空白」は折れ線上の位置比率を画面表示のデモ総距離へ換算したルート沿い推定距離です。この2つは異なる尺度で、実際の道路ネットワーク上の徒歩距離ではありません。本番用ルーティングでは道路区間ごとの距離に置き換える予定です。

地図では公式トイレを「公式オープンデータ」、休憩候補と経路を「推定デモデータ」として分けます。車椅子使用者対応情報が空欄なら「不明」とし、非対応とは扱いません。ルート近傍判定は直線距離の推定であり、実際の迂回、工事、故障、営業時間、入場制限、経路全体の車椅子通行可能性を判断しません。
