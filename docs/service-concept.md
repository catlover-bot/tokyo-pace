# サービスコンセプト

TOKYO PACE は、長時間続けて歩くことが難しい人が、休憩候補・トイレ候補・坂道の情報を比較し、「最後まで歩き切れる可能性の高い経路」を自分で選ぶための支援サービスです。最短経路の代替ではなく、判断材料を増やすことを目的とします。

## 中心指標

経路を休憩機会ごとの連続歩行区間に分け、利用者が設定した「休まずに歩ける時間」と比較します。最大連続歩行時間、次の休憩候補までの最長距離（最長休憩空白）、上限超過時間を区間データから導出し、移動を継続できる条件かを平易な日本語で示します。この判定は経路全体の安全保証ではなく、利用者が比較するための一つの判断材料です。

## MVP の前提

- 対象は新宿駅西口から東京都庁までに限定します。
- 経路・地点・設備属性はすべてプロトタイプ用の推定データであり、現地調査済みではありません。
- 区間距離・区間歩行時間と、それらから導出する移動継続可能性もデモ値です。公式データや実測値ではありません。
- 安全性、バリアフリー性、設備の利用可否を保証しません。
- 高齢者を主対象に、大きな操作領域、文字情報による凡例、地図外の一覧を提供します。
